项目名称：单词助手 (Word Assistant) - 你的私人词汇管家

【项目简介】
单词助手是一款基于 Web 的全功能英语单词学习与管理工具。它采用现代化的极简 Glassmorphism（毛玻璃）设计风格，结合科学的记忆算法，致力于帮助用户高效积累词汇量。

【核心特性】
1. 极简美学：支持平滑的深色/浅色模式切换，界面清爽。
2. 科学记忆：
   - 英译汉：四选一强化释义记忆。
   - 汉译英：拼写练习巩固书写能力。
   - 智能复习：基于错题逻辑自动安排每日学习计划。
3. 深度解析：自动检索权威词库，补全音标、释义及例句，支持真人发音。
4. 数据管理：
   - 多账号独立存储，支持云端同步（需部署服务端）。
   - 支持一键导出专属词库为 Microsoft Word (.doc) 文档。

【技术栈】
- 前端：HTML5, JavaScript (ES6+), Tailwind CSS
- 存储：LocalStorage / 自定义 Node.js 后端

【快速开始】
1. 简易模式：直接双击 `词库管理.html` 即可运行。
2. 完整模式：解压 `server.zip`，运行 `启动单词助手.cmd` 启动本地服务，享受完整翻译与同步功能。
